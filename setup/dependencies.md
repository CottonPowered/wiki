# Dependencies

### Repositories

```xml
<repositories>
    <!-- ... -->
    <repository>
        <id>cottonmc-api</id>
        <url>https://artifacts.snackbag.net/artifactory/cottonmc-api/</url>
    </repository>
</repositories>
```

### Dependencies

```xml
<dependencies>
    <!-- ... -->
    <dependency>
        <groupId>net.snackbag.cottonpowered</groupId>
        <artifactId>api</artifactId>
        <version>0.2.2+mc1.20.1</version>
        <scope>provided</scope>
    </dependency>
</dependencies>
```
